# 🚀 AI Image Caption Generator

A full-stack web app that generates AI-based captions for uploaded images with authentication support.

## 🌐 Live Demo
https://ai-image-caption-generator-phi.vercel.app

---

## ⚙️ Tech Stack

### Frontend
- React (Vite)
- Axios
- React Router
- Tailwind CSS

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Cookie-based Auth

### Deployment
- Frontend: Vercel
- Backend: Render

---

## 🔐 Features

- User Registration & Login
- Secure Authentication using HTTP-only cookies
- Cross-origin cookie handling (CORS configured)
- Image upload & AI caption generation
- Protected API routes
- Toast notifications for UX

---

## 🧠 Key Learnings

- Handling CORS in production
- Cross-origin cookies (`sameSite: "None"`, `secure: true`)
- Deploying full-stack apps (Vercel + Render)
- Managing environment variables properly
- Debugging real-world auth issues
